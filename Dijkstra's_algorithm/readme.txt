Реализация алгоритма Дейкстры для перевода математических выражений в обратную польскую нотацию на языке JavaScript.
- Для запуска js файла никаких дополнительных параметров не нужно. Строка для перевода задаётся внутри кода программы.
- Программа допускает наличие во входной строке пробельных символов. 
- Программа не допускает введение некорректного математического выражения.
- Операция возведения в степень в программе имеет левую ассоциативность. Т.е. выражение 2^2^3 = 64
