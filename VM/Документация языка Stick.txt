Stick (stck) - низкоуровневый язык программирования, предназначенный для выполнения простейших арифметических операций.


<< СИНТАКСИС ЯЗЫКА "STICK" >>

• input <ЯЧЕЙКА>
  - принимает значение с клавиатуры и складывает в ячейку памяти

• set <ЯЧЕЙКА> <ЗНАЧЕНИЕ>
  - складывает ввдённое значение в ячейку памяти

• print <ЯЧЕЙКА>
  - выводит в консоль значение, лежащее в ячейке памяти

• exit
  - команда конца программы


• add <ЯЧЕЙКА1> <ЯЧЕЙКА2> <ЯЧЕЙКА3> 
  - складывает значения <ЯЧЕЙКА1> и <ЯЧЕЙКА2> и помещает результат в <ЯЧЕЙКА3> 

• sub <ЯЧЕЙКА1> <ЯЧЕЙКА2> <ЯЧЕЙКА3> 
  - вычитает из значения <ЯЧЕЙКА1> значение <ЯЧЕЙКА2> и помещает результат в <ЯЧЕЙКА3> 

• mul <ЯЧЕЙКА1> <ЯЧЕЙКА2> <ЯЧЕЙКА3> 
  - умножает значения <ЯЧЕЙКА1> и <ЯЧЕЙКА2> и помещает результат в <ЯЧЕЙКА3> 


• cmp <ЯЧЕЙКА1> <ЯЧЕЙКА2> <ЯЧЕЙКА3> 
  - сравнивает значения <ЯЧЕЙКА1> и <ЯЧЕЙКА2>. Если они равны, возвращает 0, если нет, возвращает 1. Результат записывается в <ЯЧЕЙКА3> 

• cmpm <ЯЧЕЙКА1> <ЯЧЕЙКА2> <ЯЧЕЙКА3> 
  - сравнивает значения <ЯЧЕЙКА1> и <ЯЧЕЙКА2>. Если значение <ЯЧЕЙКА1> больше значения <ЯЧЕЙКА2>, возвращает 0, если нет, возвращает 1. Результат записывается в <ЯЧЕЙКА3>


• jmp <ЗНАЧЕНИЕ>
  - перемещает ip в заданную <ЗНАЧЕНИЕ> ячейку памяти

• jz <ЯЧЕЙКА> <ЗНАЧЕНИЕ>
  - Если значение <ЯЧЕЙКА> равно 0, перемещает ip в заданную <ЗНАЧЕНИЕ> ячейку памяти

• jnz <ЯЧЕЙКА> <ЗНАЧЕНИЕ>
  - Если значение <ЯЧЕЙКА> не равно 0, перемещает ip в заданную <ЗНАЧЕНИЕ> ячейку памяти
