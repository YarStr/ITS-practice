Программа получает на вход файл с исходной строкой, путь к файлу с выходными данными,
искомую подстроку и режим работы (brute / hash).

На выход программа выводит в файл все вхождения искомой подстроки в исходную, а также
выводит в консоль время выполнения алгоритма.

Пример ввода агрументов командной строки:
C:/>node brute_and_hash input.txt output.txt "князь Андрей" brute
C:/>node brute_and_hash input.txt output.txt "князь Андрей" hash
