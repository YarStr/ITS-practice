Реализация задания по курсу ИТС на тему "Стандарт IEEE 754"

- Запуск программы возможен в двух режимах:
  node float.js conv
  node float.js calc
  
- В первом режиме программа считывает из файла input.txt число и переводит его в нормализованный вид согласно страндарту IEEE 754.
  Результат выводится в файл output.txt
  
- Во втором режиме программа считывает из файла input.txt строку, содержащую два числа, разделенных операцией + или -.
  (!) В строке не должно быть пробельных символов (!)
  Результат выполнения арифметической операции записывается в файл output.txt во внутреннем и в десятичном виде.
